Vamos entender o que é o HTML Semântico

Quando falamos de HTML Semântico, estamos falando de tags que conseguem passar uma informação com um significado específico para o conteúdo interpretado pelo navegador, independendo apenas do texto dentro da tag para se entender o que há naquela parte do site.

Aqui veremos algumas Tags e usas respectivas utilizações

A ideia é facilitar o entendimento na hora da utilização das Tags.

  • <header>
  • <article>
  • <aside>
  • <footer>
  • <blockquote>
  • <cite>
  • <details>
  • <figure>
  • <strong>
  • <em>
  • <del>
  • <ins>
  • <sup>
  • <sub>

Uma das mais utilizadas é o <header> Cabeçalho, onde temos um grupo de suporte introdutório ou navegacional. Pode conter alguns elementos de cabeçalho mas também outros elementos como um logo, seções de cabeçalho, formulário de pesquisa, e outros.

O <article> representa um artigo com composição independente em um documento. Normalmente vemos muito sua utilização em postagem no fórum, blog e notícias.

Quando usamos <aside> estamos representando uma seção da página que pode ser considerada separada do conteúdo.

O <footer> representa um rodapé para o seu conteúdo de seção mais próximo.Normalmente um rodapé contém informações sobre o autor da seção de dados, direitos autorais ou links para documentos relacionados.

A tag <blockquote> indica que o texto incluído é uma longa citação.

Quando usamos o <cite> citamos a fonte da citação utilizada anteriormente.

Visualize o que a tag <details> faz

A tag <details> é específica para detalhes adicionais que o usuário pode abrir e fechar sob demanda.

O <figure> representa conteúdo autocontido, potencialmente com uma legenda opcional, que é especificada usando o <figcaption> elemento. A figura, sua legenda e seu conteúdo são referenciados como uma única unidade.

Ou

Utilizaremos o <strong> quando desejarmos dar destaque em algo, colocando-o em Negrito.

Utilizaremos o <em> quando desejarmos dar destaque em algo, colocando-o em Itálico.

O <del> representa uma parte do texto que foi excluída de um documento. Quando utilizado, geralmente os navegadores traçam uma linha no texto que foi excluído.

A tag <ins> representa uma parte do texto que foi inserido em um documento.Qando utilizada, geralmente os navegadores sublinham o texto inserido.

A tag <sup> define um texto sobrescrito. Quando utilizarmos essa tag o texto aparecerá meio caractere acima da linha normal. Dica: Pode ser utilizado para notas de rodapé.Exemplo:[1]

Um dos grandes motivos para nos preocuparmos com a semântica que usamos no site vem das ferramentas de indexação de buscadores, que colocam os sites mais semânticos e estruturados como prioridade nas respostas das buscas. A outra grande preocupação é com as ferramentas de acessibilidade, que permitem que pessoas com deficiência, por exemplo pessoas cegas, consigam usar um site através de softwares leitores de tela de maneira padronizada e sem problemas. Uma coisa que precisamos lembrar é que devemos escolher as tags pelo o que elas representam e não como elas são mostradas na tela do navegador. Estilização deve ficar no CSS e estrutura no HTML.
